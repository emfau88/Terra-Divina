# Terra Divina / Mini World God Game

This is the active modular Phaser 3 + Vite + TypeScript implementation.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current Scope

Implemented systems include:

- main menu
- new world setup
- scenario selection
- seeded world generation
- humans, orcs, elves, dwarves
- villages, buildings, roads, build sites
- unit roles and AI
- resources, hunger, fire, diplomacy, goals
- god tools and terrain tools
- wolves and demons
- inspect panel and event feed
- single-slot localStorage save/load
- mobile HUD/tool dock

## Current Priorities

1. War/tension readability.
2. Save/load hardening.
3. Scenario balance.
4. Visual consistency.
